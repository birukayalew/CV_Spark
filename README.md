ResuMind is an **AI-powered resume analyzer** built with **React**, **React Router**, and **Puter.js**. It enables users to upload, store, and analyze resumes directly in the browser - **no backend required**. Using smart AI evaluations, ResuMind matches resumes to job listings, provides **ATS-style scores**, and generates **custom, actionable feedback** tailored to each role.
